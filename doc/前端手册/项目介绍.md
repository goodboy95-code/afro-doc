# 项目介绍

## 简介

`afro-vue3` 是一个基于`Vue3.0`、`Vite`、`ElementPlus`、`TypeScript`的后台系统前端模版。
目前包括动态菜单、权限校验、按钮级别权限控制等功能。
该项目可以作为后台项目的启动模版，以帮助你快速搭建企业级后台产品。
也可以作为一个示例，用于学习`vue3`、`vite`、`ts`等主流技术。

## 项目结构

```markdown
├── src
    ├─ assert // 静态资源
    ├─ axios // axios http请求和数据类型
    ├─ components // 组件树
    ├─ data // 组件需要数据
    ├─ pinia // 全局数据管理
    ├─ router // 路由处理
    └─ utils // 工具库
```
