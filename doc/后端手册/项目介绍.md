# 项目介绍

## 简介

该项目是一个基于Spring Boot的后台系统后端模版。

## 项目结构

- 项目结构遵循Maven标准目录结构。
```markdown
├─ annotation // 注解目录
├─ aop // 切面目录
├─ async // 异步任务目录
├─ conf // 配置文件
├─ constant // 常量目录
├─ controller // 控制层
├─ entity // 实体类
├─ enums // 枚举类
├─ exception // 异常处理目录
├─ filter // 过滤器目录
├─ mapper // MyBatis Mapper目录
├─ security // 安全相关目录
├─ service // 服务层
└─ utils // 工具类目录
```

## 技术栈

- Spring Boot
- MyBatis
- MySQL
- Redis
- Hutools
